# WEIZAC

WEIZAC computer history and simulator project.

## Quick Start
1. Open in VS Code
2. Activate env: `.\env\Scripts\activate`
3. Run `@autoagent /run` in Copilot Chat

## Structure
- `program.md` — AutoAgent task definition (you write this)
- `reference/` — historical documents and specs
- `state/` — runtime knowledge and plans
- `autoagent.config.json` — iteration and tool config
